## Descripcion

Este repositorio contiene el manual de operaciones de Vanity Salon & SPA, fundado en 2017 por Alejandra Ponce. El manual esta disenado para guiar al equipo en los procesos, protocolos y estandares que definen la experiencia Vanity.
